Where is the unlocker directory?

Why does the User Generated folder take over lib folder?

So the unlocker has a space to keep your generated files including what prestige and items you selected to use


How do I remove the User Generated folder?

Head into the unlocker directory and remove the folder

Installation

  1. Download the unlocker from the official website

  2. Extract the .rar file and run the Loader as Administrator

  3. Accept the Discord authorization that opens in your web browser

  4. Go to the 'Market' tab

    1. In the 'Profile' section, select the prestige level and item amount you want
    2. Click 'Select Path' and enter your Paks folder location
    3. After that, click 'Generate' and wait for the process to finish

  5. Go to the 'Unlocker' tab

    1. In the 'Config' section, choose your settings
    2. Click 'Start' and press 'Yes' on the Fiddler root certificate showing up
    3. After accepting the Fiddler Root, start your game


    Your cookies will be detected automatically when you start your game




Troubleshoot

No internet connection on entire PC

Disable the proxy that was enabled after starting the Unlocker


No Network Connection

Playing on Steam requires an SSL Bypass - Learn more here: Requirements


Other causes might be:

  1. No internet connection - Make sure you have a working internet connection

  2. You are banned

Invalid bhvrSession

Enter the correct bhvrSession


Initialization Error

Keep all the files from the VOID folder together


The input string '0.0' was not in a correct format

Select an amount for Prestige and Items - Minimum of 2


Perks, Addons, and Items resetting after game

Simply open the Bloodweb, then open the loadout and switch the presets back and forth



Failed to create Fiddler root certificate

Click 'Yes' when the Fiddler root certificate prompt appears

If the certificate no longer appears, go to the unlocker directory and delete the 'cert' folder


Unexpected Parameters Error

Delete the unlocker directory, then start the loader again and enter the correct path when prompted


Save Game Error

Verify your game files
